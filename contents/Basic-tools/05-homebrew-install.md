## Homebrew

Homebrew is a simple tool that acts like an App Store for the command line. It helps you easily download and install other necessary technical programs on your Mac.
Homebrew是一个简单的工具，它就像命令行世界的应用商店。有了它，您就可以方便地在 Mac 电脑上下载和安装其他需要的技术程序。

Click <https://brew.sh/> or search homebrew in your browser and click the copy button to copy the installation command.
点击这个链接 [https://brew.sh/](https://brew.sh/%20) 或者在您的浏览器搜索 homebrew 并点击复制按钮去复制下载homebrew的下载命令。

![](./images/Toolkit/image25.png)

Open your Terminal.
打开您的命令行窗口（Terminal）。

![](./images/Toolkit/image26.png)

Copy and paste the following entire command into the terminal window.
复制并粘贴下面整条命令到命令行窗口中。

> Note: This command tells the computer to automatically fetch the installation script for Homebrew from the internet and run it.
> 注意：这条命令的作用是告诉电脑自动从互联网上抓取 Homebrew的安装脚本并运行它。

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

![](./images/Toolkit/image27.png)

Enter your Mac password when prompted (the characters won't appear as you type, which is normal).
当系统提示时，输入您的 Mac电脑锁屏密码（输入时屏幕上不会显示字符，这是正常的）。

![](./images/Toolkit/image320.png)

Press Enter.
按下回车键。

![](./images/Toolkit/image330.png)

Wait for the installation to finish.
等待安装完成。

![](./images/Toolkit/image340.png)
